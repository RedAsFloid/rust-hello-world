Just a simple hello world with Rust and Docker

## Requirements
 - Docker
 - Docker Compose

## Running the project

```
docker-compose build
docker-compose up
```

### Will be shown

```bash
hello_1  | Hi ! The project is working
hello_1  | Lets see some basic informations
hello_1  | Kevin Martins
```
